Ruta recomendada:

· ud2/actividades/ud2_act1/js

Crear estructura (Windows PowerShell):

· cd C:\ruta\que\quieras
· mkdir ud2
· cd ud2
· mkdir actividades
· cd actividades
· mkdir ud2_act1
· cd ud2_act1
· mkdir js

CONFIGURAR TYPESCRIPT

Inicializar TypeScript:

· npx tsc --init

En tsconfig.json (dentro de ud2_act1) ajustar:

· "module": "CommonJS"
· "outDir": "./js"
· "rootDir": "./"

ARCHIVOS TYPESCRIPT

Todos en la carpeta: ud2/actividades/ud2_act1

· IAlumno.ts
· Alumno.ts
· AlumnoFP.ts
· Clase.ts
· alumnos_fp.ts

(El compilado .js se generará dentro de js/ al ejecutar npx tsc)

INTERFAZ IALUMNO

Objetivo: definir la forma básica del alumno (datos + métodos).

Archivo: IAlumno.ts

Ideas clave:

· Es solo una “plantilla” de propiedades y métodos.
· Las clases que implementen IAlumno deben cumplir estas firmas.

CLASE ALUMNO

Objetivo: implementar IAlumno, gestionar id autoincremental, lógica de aprobado.

Archivo: Alumno.ts

Puntos importantes:

· static contador → lleva la cuenta de alumnos creados.
· readonly id → no se puede modificar tras el constructor.
· getTotalAlumnosCreados() → devuelve cuántos se han creado.

CLASE ALUMNOFP (HERENCIA)

Objetivo: extender Alumno con más propiedades y comportamiento extra.

Archivo: AlumnoFP.ts

Puntos importantes:

· extends Alumno → hereda propiedades y métodos.
· super(...) → llama al constructor de la clase base.
· override mostrarInfo → añade info de ciclo y módulo.
· recomendarEspecialidad → lógica específica de FP.

CLASE CLASE (GESTOR DE ALUMNOS)

Objetivo: gestionar un grupo de alumnos en un array privado.

Archivo: Clase.ts

Puntos importantes:

· private alumnos → encapsulación.
· mostrarTodos → devuelve todo el array.
· aprobados → filtra por nota >= 5 usando estaAprobado.
· buscarPorNombre → filtro por substring en el nombre.
· totalAlumnos (static) → delega en Alumno.getTotalAlumnosCreados().

PROGRAMA PRINCIPAL (alumnos_fp.ts)

Objetivo: crear alumnos, añadirlos a clase, mostrar info y estadísticas.

Archivo: alumnos_fp.ts

Puntos importantes:

· Se mezclan Alumno y AlumnoFP en el mismo array (polimorfismo).
· instanceof AlumnoFP → para usar recomendarEspecialidad.

Se muestran:

· Todos los alumnos.
· Solo los aprobados.
· Búsqueda por nombre.
· Total de alumnos creados.
· Recomendación de especialidad para FP.

COMPILAR Y EJECUTAR

Compilar TypeScript:

· npx tsc

Ejecutar con Node:

· node js/alumnos_fp.js

Si hay error de “Cannot use import statement outside a module”:
Asegurarse de que en tsconfig.json está:

· "module": "CommonJS"
· "outDir": "./js"
· "rootDir": "./"
